\# Replication Package



This repository contains the replication package for the thesis:



"Do LLMs Improve in Modelling Tasks with Their Evolution? An Empirical Study"



\## Contents



\- Raw model outputs (.jsonl)

\- Evaluation results (.csv)

\- Datasets

\- Evaluation scripts



\## Datasets



\- Synthetic dataset

\- RepairVision dataset



\## Model Families



\- GPT

\- Gemini

\- DeepSeek

\- Llama

\- Qwen



\## Running Evaluation



Example:



python custom\_eval\_paper\_like.py INPUT.jsonl OUTPUT.csv
